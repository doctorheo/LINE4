# 4호선 LCD 프로젝트
![이미지](/동작/진접행/2.PNG)
## 참고사항
- 제미나이 써서 만들었어요. 이상한 부분 많을겁니다.
- 다음 화면으로 넘어가거나 글씨가 바뀌는 데 최대 5초가 걸립니다. 버튼을 눌러도 화면이 바로 바뀌지 않습니다.
- 행선지는 진접행/사당행만 존재합니다. 행선지 추가는 할 계획이 전혀 없습니다.

## 진행도
- 사당역, 총신대입구(이수)역, 동작역, 이촌역, 진접역 만들어뒀습니다. 이촌역에서 다음역 누르면 진접역으로 바로 넘어갑니다.
- 선택 화면은 거의 다 작동하는 상태이나, 자잘한 버그가 많습니다.

## 설치/실행 방법
1. 파이썬을 설치하십시오.
2. 파이썬에 Tkinter, Pillow, Pygame 라이브러리를 설치하십시오.
3. 이 페이지에서 Code-Download ZIP을 눌러 다운받은뒤, 압축을 풀고 subway_system.py를 실행하십시오.

## 사용 방법
- F키를 눌러 전체 화면을 사용하십시오.
- 표시화면에서 행선지 글씨 위치가 안맞을 때에는 화살표 키를 사용하여 옮기십시오.
- 선택 화면에서 출발화면/방송 버튼은 한번 누르면 화면이 뜨고, 한번 더 누르면 방송이 나오며, 또 한번 더 누르면 원래대로 돌아옵니다. 출발화면 상태에서는 다른 버튼이 작동하지 않습니다.
- 대기 버튼은 한번 누르면 노선도 화면이 뜨고, 한번 더 누르면 원래대로 돌아옵니다. 대기 화면에서 역을 전환하지 마십시오.
- 행선지변경 버튼을 눌러 진접행/사당행 간에 전환하십시오.
- 한글을 읽으실 수 있으시다면 나머지 기능은 알아서 쓰실 수 있을겁니다.
